# 嘘発見モード

スマホでターゲットに渡して使う、心理的プレッシャー演出付きの嘘発見アプリです。

## 起動方法

```bash
npm install
npm run dev
```

スマホからアクセスする場合は、PCと同じWi-Fiに接続し、表示されるネットワークURLを開いてください。

## 使い方

1. **警告画面** — 3つのルールを読んで「了解して次へ進む」
2. **キャリブレーション** — 3つの簡単な質問にボタンを押しながら声で回答
3. **準備画面** — ターゲットにスマホを渡し「測定開始」をタップ
4. **本番質問** — カウントダウン後、ボタンを押しながら声で回答
5. **解析中** — 演出付きの解析画面
6. **結果** — 嘘の確率が表示される

## 質問文の変更

画面上部の ⚙️ ボタンから本番質問文を変更できます（ターゲットには見えない位置にあります）。

## スマホにインストール（PWA）

HTTPS で公開された URL（Vercel 等）にアクセスしてインストールできます。

### Android（Chrome）
1. サイトを開く
2. 画面下部の「インストール」バナーをタップ
3. または Chrome メニュー →「アプリをインストール」

### iPhone（Safari）
1. Safari でサイトを開く（Chrome では不可）
2. 左下の 📲 ボタンをタップ
3. 共有ボタン →「ホーム画面に追加」→「追加」

インストール後はホーム画面のアイコンからアプリのように起動できます。

## 嘘判定ロジック

- ボタンを押してから喋るまでの遅延
- 喋り終わった後にボタンを離すまでの遅延（ホールド不一致）
- 応答までのタイムラグ
- 発声時間の長さ

これらを組み合わせて嘘の確率を算出します。
